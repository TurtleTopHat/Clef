Pressing start presents uder with options and brings up the piano display.

The following options are then presented to the user and the piano is made visible to the user. Clicking on the piano keys produces the relevant note.

The user can choose the note which they wish the chord progression to be genrated from.

The user can then choose which accidental to apply.

The user is presented with the option to choose either the major or minor scale.

If the user clicks 'See Results' the chord progression is genrated.

After clicking see results the following chords are produced and produce sounds when clicked on.

When a user clicks on a chord it is highlighted on the piano and its sound is played.
